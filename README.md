# Script Reader 3000

A web app for reading scripts word-by-word with customizable settings.

## Features
- Paste any script and read it word-by-word
- Keyboard controls for seamless navigation
- Adjust font size (16-32px)
- Dark/Light mode toggle
- Configurable context words (3/5/10 words around current)
- Auto-save scripts to browser storage

## How to Use

1. Open `index.html` in your web browser
2. Paste your script in the text box
3. Click **Start** to enter reading mode
4. Use keyboard controls to navigate

### Keyboard Controls
- **Space** - Next word
- **Backspace** - Previous word
- **R** - Restart from beginning
- **Esc** - Exit reading mode

### Settings
- Adjust font size with the slider
- Toggle between Dark and Light mode
- Choose how many surrounding words to display (3, 5, or 10)

All settings and your last script are automatically saved in browser storage!

## Files
- `index.html` - Main HTML structure
- `styles.css` - All styling with dark/light mode support
- `script.js` - JavaScript functionality and state management

## Browser Support
Works in all modern browsers (Chrome, Firefox, Safari, Edge)
